# 🎨 ColorMixer

Color playground with HSL picker, multi-stop gradient builder, and saved palette.

## Features
- **Picker** — HSL sliders → HEX / RGB / HSL / OKLCH output, one-click copy
- **Gradient** — linear / radial / conic, 3 color stops, angle control, CSS export
- **Palette** — save colors, export JSON or CSS variables
- LocalStorage persistence

## Run
Open `index.html` directly in a browser.
